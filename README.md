# Tetris

パステルカラーのテトリスWebアプリです。ビルド不要のVanilla JS実装で、ブラウザで即プレイできます。

**[▶ Play Now](https://yaichi-240.github.io/tetris-app/)**

---

## 特徴

- **3段階の難易度** — EASY / NORMAL / HARD（落下速度・加速率が異なる）
- **ゴーストピース** — 着地位置を透過表示
- **ネクストピース** — 次のピースをプレビュー
- **ハイスコア管理** — 難易度ごとにTop5をローカル保存（localStorage）
- **モバイル対応** — タッチボタン操作・レスポンシブレイアウト
- **パステルカラーUI** — 明るく見やすいデザイン

---

## 操作方法

### キーボード

| キー | 操作 |
|------|------|
| `←` / `→` | 左右移動 |
| `↑` | 回転 |
| `↓` | ソフトドロップ |
| `Space` | ハードドロップ |
| `Esc` | ポーズ |

### タッチ（モバイル）

画面下部のタッチボタンで操作します。

---

## 難易度設定

| モード | 初期落下速度 | ソフトドロップ |
|--------|-------------|----------------|
| EASY   | 800ms/段    | 5倍速          |
| NORMAL | 500ms/段    | 10倍速         |
| HARD   | 200ms/段    | 20倍速         |

レベルが上がるごとに落下速度が上昇します。

## ファイル構成

```
tetris-app/
├── index.html
├── style.css
└── js/
    ├── config.js      # 難易度設定・定数
    ├── tetromino.js   # ピース定義・回転・バッグ生成
    ├── board.js       # ボード状態・DOM描画・ゴーストピース
    ├── score.js       # スコア計算・localStorage管理
    ├── input.js       # キーボード・タッチ入力・DAS
    └── main.js        # ゲームループ・画面制御
```

## ローカルで実行

```bash
git clone https://github.com/yaichi-240/tetris-app.git
cd tetris-app
# index.html をブラウザで開くだけで動作します
```
